Privacy & Impressum

This page provides transparency about who runs this website, what data is processed, and how to get in touch.

About this site

This is a free, non-commercial, open-source educational resource for people preparing for the German citizenship test (Einbürgerungstest / Leben in Deutschland). It is hosted on GitHub Pages.

The site does not sell anything, does not run advertising, does not require registration, and does not store any personal user data.

Operator / Impressum

This site is operated as a personal hobby project by an individual based in Germany.

For any questions, corrections, takedown requests, or legal notices, please open an issue on the project's GitHub repository:
github.com/abdullahbutt/german-citizenship-test-english-urdu/issues

Issues are publicly visible by default. If you need a private channel, please mention this in your issue and a private email will be exchanged.

Data we process

We use Cloudflare Web Analytics, which is a privacy-respecting analytics service:

  • It does not use cookies or any other client-side storage.
  • It does not track users across sites.
  • It does not fingerprint browsers or build user profiles.
  • It collects only aggregated, anonymous metrics: page URL, country, referrer, browser, device type, and timing.
  • IP addresses are processed transiently and not stored.

Because no personal data is collected, no consent banner is required under the EU GDPR (Art. 6) or the German TTDSG (§ 25). Cloudflare's analytics privacy details: cloudflare.com/web-analytics.

Local browser storage

The site uses your browser's localStorage only to remember your preferences:

  • Your chosen language (English or Urdu)
  • Your chosen theme (light or dark mode)
  • Whether you've dismissed the "Install as app" hint

This data stays on your device and is never sent to any server. You can clear it any time via your browser settings.

Service Worker (offline mode)

If your browser supports it, the site installs a service worker that caches pages on your device so you can read them without an internet connection. Cached files stay on your device only and are never transmitted elsewhere.

External links and resources

The site links to and embeds resources from:

  • bamf.de — official BAMF question catalog and online test center
  • jsDelivr CDN — Bootstrap CSS framework
  • Google Fonts — Inter and Noto Nastaliq Urdu fonts
  • GitHub Pages — hosting infrastructure
  • Cloudflare — anonymous analytics

These third parties may receive standard technical information from your browser (such as your IP address) when their resources are loaded. Each operates under its own privacy policy.

Source content and copyright

The German question text is taken from the official BAMF question catalog for the Leben in Deutschland / Einbürgerungstest, which is publicly available material. The English and Urdu translations and explanations are original contributions provided freely for educational use.

This site is not affiliated with, endorsed by, or connected to BAMF, the German government, or any official authority.

Your rights

Because no personally identifiable data is stored, there is no personal record to access, rectify, or delete. If you believe data about you is being processed in error, please open a GitHub issue and it will be addressed.

Changes to this notice

If anything material changes (for example, if a new analytics tool is added), this page will be updated and the date below will reflect the change.

Datenschutz & Impressum

Diese Seite informiert transparent darüber, wer diese Website betreibt, welche Daten verarbeitet werden und wie eine Kontaktaufnahme möglich ist.

Über diese Website

Dies ist ein kostenloses, nicht-kommerzielles Open-Source-Lernangebot für Personen, die sich auf den deutschen Einbürgerungstest bzw. den Test Leben in Deutschland vorbereiten. Die Seite wird über GitHub Pages bereitgestellt.

Auf der Seite werden keine Produkte verkauft, keine Werbung geschaltet, keine Registrierung verlangt und keine personenbezogenen Nutzerdaten gespeichert.

Betreiber / Impressum

Diese Seite ist ein privates, nicht-kommerzielles Hobbyprojekt einer in Deutschland ansässigen Privatperson.

Anfragen, Korrekturen, Beschwerden oder rechtliche Hinweise bitte über das GitHub-Issue-System des Projekts einreichen:
github.com/abdullahbutt/german-citizenship-test-english-urdu/issues

Issues sind standardmäßig öffentlich sichtbar. Wird ein vertraulicher Kanal benötigt, bitte dies im Issue erwähnen — eine private E-Mail-Korrespondenz kann dann eingerichtet werden.

Verarbeitete Daten

Die Seite nutzt Cloudflare Web Analytics, einen datenschutzfreundlichen Analysedienst:

  • Es werden keine Cookies und keine clientseitigen Speichermechanismen verwendet.
  • Es findet kein seitenübergreifendes Nutzer-Tracking statt.
  • Es werden keine Browser-Fingerabdrücke erstellt und keine Nutzerprofile gebildet.
  • Erfasst werden nur aggregierte, anonyme Metriken: Seiten-URL, Land, Referrer, Browser, Gerätetyp, Ladezeit.
  • IP-Adressen werden nur kurz transient verarbeitet und nicht gespeichert.

Da keine personenbezogenen Daten erhoben werden, ist gemäß DSGVO Art. 6 und TTDSG § 25 keine Einwilligung (Cookie-Banner) erforderlich. Details: cloudflare.com/web-analytics.

Lokaler Browser-Speicher

Die Seite nutzt localStorage ausschließlich, um Ihre Einstellungen zu merken:

  • Gewählte Sprache (Englisch oder Urdu)
  • Gewähltes Farbschema (hell oder dunkel)
  • Ob der Hinweis „Als App installieren" geschlossen wurde

Diese Daten verbleiben ausschließlich auf Ihrem Gerät und werden zu keinem Zeitpunkt an einen Server übertragen. Sie können sie jederzeit über die Einstellungen Ihres Browsers löschen.

Service Worker (Offline-Modus)

Sofern der Browser dies unterstützt, registriert die Seite einen Service Worker, der Inhalte lokal zwischenspeichert. So sind die Inhalte auch ohne Internetverbindung lesbar. Die zwischengespeicherten Dateien verbleiben auf Ihrem Gerät und werden nicht weitergegeben.
