## 🖼️ تصاویر والے سوالات

کچھ سوالات میں تصاویر شامل ہیں (نشانات، نقشے، جھنڈے)۔ اصل تصاویر دیکھنے کے لیے [BAMF انٹرایکٹو ٹیسٹ سینٹر](https://oet.bamf.de/ords/oetut/f?p=514:1:0) استعمال کریں۔

**تصاویر والے عمومی سوالات:**

| سوال | تصویر میں |
|---|---|
| سوال 21 | 4 نشانات — جرمنی کا نشان پہچانیں (سنہرے پس منظر پر سیاہ عقاب) |
| سوال 160 | جرمنی کا نقشہ — مشرقی/مغربی تقسیم |
| سوال 163 | جرمنی کی سرحدوں کا نقشہ |
| سوال 177 | برلن کا نقشہ — 4 اتحادی علاقوں میں تقسیم |
| سوال 195 | تقسیم شدہ جرمنی / قبضے کے علاقوں کا نقشہ |

**ریاستی تصاویر والے سوالات (ہر ریاست):**

| سوال | تصویر میں |
|---|---|
| سوال 301 | 4 نشانات — اپنی ریاست کا نشان پہچانیں |
| سوال 302 | جرمنی کا نقشہ — کون سی ریاست نمایاں ہے |
| سوال 304 | نمبر والا نقشہ — اپنی ریاست کا نمبر پہچانیں |

## 🆕 تازہ ترین سوالات (جولائی 2024)

10 سوالات کو نئے سوالات سے بدل دیا گیا جو **جرمنی میں یہودی زندگی، اسرائیل کے وجود کا حق، اور سام دشمنی** سے متعلق ہیں (7 اکتوبر 2023 کے واقعات کے جواب میں)۔ ان پر فائلوں میں *(جولائی 2024 میں تازہ کاری)* کا نشان لگا ہے:

| # | نیا موضوع | درست جواب |
|---|---|---|
| سوال 14 | اظہار رائے کی حدود (اب حماس/داعش کی علامات کا ذکر) | انٹرنیٹ پر رائے کا اظہار کر سکتے ہیں |
| سوال 59 | جرمنی کی سب سے بڑی یہودی جماعتیں | برلن اور میونخ |
| سوال 66 | جرمنی میں یہودی زندگی کب سے ہے؟ | 1,700 سال سے زیادہ |
| سوال 96 | ہولوکاسٹ سے انکار کی سزا | 5 سال تک قید یا جرمانہ |
| سوال 111 | اسرائیل کے حوالے سے ممنوعہ اعمال | اسرائیل کی تباہی کا عوامی مطالبہ |
| سوال 118 | مکابی کھیل کلبوں کا رکن کون بن سکتا ہے؟ | تمام لوگ |
| سوال 149 | سام دشمنی کی مثال | ہولوکاسٹ سے انکار |
| سوال 182 | یہودی عبادت گاہ کا نام | عبادت خانہ (Synagoge) |
| سوال 184 | اسرائیل کی قانونی بنیاد | اقوام متحدہ کی قرارداد |
| سوال 206 | "Stolpersteine" کس کی یاد میں ہیں؟ | نازی دور کے متاثرین |
| سوال 288 | جرمنی کی اسرائیل کے لیے خاص ذمہ داری | نازی جرائم |
