# Contributing

Thank you for helping improve these citizenship test materials! Contributions of all kinds are welcome — from fixing a typo to flagging an outdated answer.

## Ways to contribute

- **Fix errors** — wrong answers, incorrect translations, or outdated information
- **Improve translations** — better English or Urdu wording for questions or explanations
- **Flag outdated answers** — officeholders (Chancellor, Minister-Presidents) change with elections; please report if an answer is no longer correct
- **Add missing explanations** — if a question lacks context or background, improvements are welcome
- **Report issues** — something confusing or missing? Open an issue and describe it

## How to contribute

1. **Fork** this repository
2. **Select the correct branch** — `english` for English content, `urdu` for Urdu content
3. **Make your changes** in your forked copy
4. **Submit a Pull Request** with a clear description of what you changed and why

For small fixes (typos, broken links), a Pull Request is enough. For larger changes, please open an **Issue** first to discuss before investing time in it.

## Guidelines

- Always reference the official BAMF question number (e.g. Q14, Q96) when reporting or fixing an answer
- Keep the bilingual format consistent — each question should have both German and the branch language (English or Urdu)
- For outdated officeholder answers, link to an official source confirming the change
- Maintain the existing file naming convention (`questions-001-050.md`, state files like `bayern.md` etc.)

## Reporting outdated answers

This is the most valuable contribution you can make. If you spot an answer that has changed due to an election or policy update:

1. Open an **Issue** with the title: `Outdated answer: Q[number]`
2. Include the question number, current answer in the repo, and the correct updated answer with a source link
